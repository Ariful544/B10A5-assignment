Important FAQ.

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for representing HTML, XML, and other structured documents as a tree of nodes. Each node represents a part of the document, such as an element, attribute, text, or comment. The DOM allows JavaScript to interact with the document and manipulate its content, structure, and style.

How do you select an element from the DOM?

When we want to select an element from the DOM we can use various methods such as getElementById, getElementsByClass, getElementsByTagName, querySelector and querySelectorAll.For instance, document.getElementsById().innerHTML

How do you manipulate an elements attributes and styles using the DOM?

The DOM provides methods and properties to access and modify an element's attributes and styles.Such as getAttribute, setAttribute, for image src,style and classList. By using these methods and properties, we can effectively manipulate an element's attributes and styles to create dynamic and interactive web pages.

What is event delegation in the context of the DOM,and why is it useful?

Event Delegation in the context of the DOM is a technique where you attach an event listener to a parent element instead of directly to each child element. This allows you to handle events for multiple child elements efficiently, especially when the number of child elements is dynamic or large.
